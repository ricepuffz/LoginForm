Search

The functionality of the search on the main page is different than that of that on the navigation bar.

On the main page, you select the type of information you want with the drop down selection, type in the symbol name in the search box itself and search.

On the navigation bar, there is no drop down selection. The syntax here is symbol:type

However, it is not necassary to specify a type (when using the navigation bar), when no type is appended to the symbol the standard type being used is "volume". When searching using the navigation bar searchbox from a stock information page, the currently viewed type is being used automatically.


API Usage

All stock data delivered on this page is being retrieved using the Alphavantage API. Since only a free plan is being used, the API call frequency is 5 per minute and 500 per day.

Because of this limit, a minute has to pass after opening a stock information page until a new stock information page can be opened, since opening one requires 4 API calls.

If you open a new stock information page before a minute has passed, simply wait a minute and then click the "Reload page" button on the error page or return to the main page by using the link.
